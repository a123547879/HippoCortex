## 🧠 项目简介
本项目模拟**人脑海马体+大脑皮层**的工作机制，打造了专为大模型设计的智能长期记忆系统。
摒弃传统硬编码规则，通过**动态专家路由、赫布学习、突触修剪、睡眠巩固**等类脑机制，实现知识的自动分类、智能检索、持续学习与自然遗忘。

已完成V6.3核心修复，**路由准确率95%+**，支持百万级记忆存储，可直接落地为个人AI助理、知识库助手的记忆核心。

## ✨ 核心特性
### 🔹 类脑核心机制
- **海马体智能路由**：基于向量相似度的专家分配，彻底解决路由偏向问题
- **动态专家分区**：概念/空间/抽象/视觉四大专家，知识分类存储更合理
- **赫布联想学习**：突触连接自动强化，实现记忆间的自主关联
- **睡眠巩固机制**：夜间修剪弱连接，优化记忆结构，模拟人脑遗忘规律
- **稀疏编码存储**：极低内存占用，高效存储海量记忆

### 🔹 工程级能力
- 兼容Ollama本地Embedding/大模型，无需云端API
- FAISS向量索引自动修复，数据安全不丢失
- 记忆自动分层（短期/长期/永久），支持批量重分配
- 毫秒级检索响应，支持持续在线学习
- 完整日志+状态监控，可视化突触学习热力图

## 🎯 解决的核心痛点
1. ❌ 传统RAG：知识混杂、检索无关、无法分类 → ✅ 专家分区，精准路由
2. ❌ 静态记忆：无法学习、无法遗忘 → ✅ 在线学习+自然衰减
3. ❌ 无类脑逻辑：只是向量检索 → ✅ 赫布学习+突触关联，真正类脑
